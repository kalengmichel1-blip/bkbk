# Team BKBK - Website Redesign

The official website for **Team BKBK** (Barnab√© Kikaya Bin Karubi), focusing on sovereignty, memory, and political thought.

## Features
- **Modern Tech Stack**: Built with Next.js 15 (App Router), React, and Tailwind CSS.
- **Dynamic Archives**: Full content migration with category filtering.
- **Interactive Visuals**: Custom 3D Globe (`three.js`) and animated Splash Screen (`framer-motion`).
- **Brand Identity**: Faithful implementation of the BKBK blue and red color scheme.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
